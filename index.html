What is RhythMIR?

RhythMIR is a project by Craig Jeffrey which uses techniques of Music Information Retrieval to generate metadata files for music tracks. RhythMIR is also a simple rhythm game with gameplay generated from these metadata files. RhythMIR is currently in development as Craig's Honours Project for the last year of Computer Games Technology at Abertay University.

Quick Install

Download RhythMIR.zip using the latest releases button on the bar at the top of the page. You can then run RhythMIR.exe from this folder. Note that no songs are included with the download, you must add your own. They must be .wav format.
